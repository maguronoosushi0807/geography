# 基本
- 与えられた図のすべての情報をみておく
- 正誤判定は一文ずつ独立して判断
- 明確な誤りから消す
- 明確な偏り方をしている分布から特定していく
- グラフ内で特別に与えられたものから処理する
- 値でわからない → 変化を見る
- 変に逆張らない

# 自然環境
### 火山
- 火山 → 新期造山帯のプレート境界 or アフリカ大地溝帯
- 火山噴出物 → 隙間が多く、水はけが良い
### 氷河
- 山岳氷河 → 標高が基準を超えるかで判断 赤道付近:5000m-6000m/温帯:3000m-4000m
- 氷河 → 夏に融解して川の流量 増加
- アルプス山脈 → 氷河
### 自然災害
- 震源の深さ
  - 直下型 → 浅い
  - 海溝型 → 深い
- 中米 → ハリケーンが襲来する
- 洪水の発生時期 → 川の流量変化の多い高緯度帯に着目
### 雨温図
- 気温/降水量の図の読み方
  - 1.気温 → 北半球/南半球/赤道に分類
  - 2.降水 → 内陸/海岸に分類
  - 3.Csなど、問題によっては気候から位置まで推論
- 高山地域の気温の較差 → 年較差 < 日較差
- 日照時間 → 雨の量から判断
### 植生
- 土壌/植生 → 気候
- 森林の有無 → 気温+降水量
- 植物の分解 → CO2とH2Oになると考える
### 知識
- 変動帯 → 新期造山帯
- 高低 → プレート境界
- 熱帯収束帯の動く幅は年によってまちまち
- 卓越風 = その時期に一番吹きやすい風
- エルニーニョ現象 = 原因不明の貿易風の弱化により湧昇流が減ることで起こる、ペルー沖の海水温の異常上昇
- 積雪 → 12月から(北海道は例外的にもっと早い)

# 産業
- 文字式で仮想的な計算をしてみる
- 日本の投資先 → 1970:アジアNIEs/1980後:ASEAN/その後:中国

- 耕地 = 田/畑/果樹園など、農業用地

- 農業のGDP割合 = 工業化の進んでいなさ
- 農業の特徴

  - 西アジア → 灌漑
  - ヨーロッパ → 混合農業
  - アフリカ → 焼畑農業
- 作物分布 → 絞りにくいなら育たない部分を見る
- 豚 → イスラム教が多い国に着目
- 狭い国 → 土地生産性が高くなりがち
- 米
  - 生産 → アジアに9割が集中
  - 輸出 → タイ・パキスタンが上位
  - 輸入 → 東南/南アジアに加え、西アジア/ギニアなどが意外と多い
- 生鮮食品/野菜 → 輸送技術の発達で輸入されてきている
- 牛乳の生産 → 北海道/大都市近郊
- BSEの受けてトレーサビリティが整備された
- 遺伝子組み換え → 除草剤/害虫に耐える → 雑草を枯らすために薬を使う
- 遺伝子組み換え → 輸入されている(表示義務あり)

### 鉱業
- 資源の最大の生産国/消費国が同じ物

- バイオマス → ブラジルのサトウキビから始まった
